MEUEXAME — PROTÓTIPO HTML

Como abrir:
1. Extraia o arquivo ZIP.
2. Abra o arquivo index.html no navegador.
3. Navegue pelos botões e links entre as páginas.

Páginas:
- index.html: home pública e consulta
- resultado.html: resultado encontrado
- ajuda.html: central de ajuda
- privacidade.html: política de privacidade
- login.html: acesso da equipe
- dashboard.html: visão geral administrativa
- exames.html: listagem de exames
- novo-exame.html: cadastro manual
- detalhes-exame.html: detalhes e histórico
- usuarios.html: usuários da equipe
- logs.html: auditoria do sistema
- design-system.html: tokens, componentes e padrões visuais

Tecnologias:
- HTML5
- Tailwind CSS via CDN
- JavaScript puro
- Lucide Icons via CDN

Observação:
Este é um protótipo visual. As ações são simuladas e não existe backend ou banco de dados.
